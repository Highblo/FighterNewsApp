## アプリケーション名


### 格闘技ファンの為のNEWSサイト


## アプリケーション概要


格闘技が好きな人の為のNewsサイトです。


## 要件定義

### 目的

・格闘技に関するニュースを素早く情報収集する。

### 概要

1.1～10の数字でログインする。

2.好きな格闘技を入力して記事を検索する。


### 使用技術

・React
・Typescript
・ChakraUI
・AtomicDesign
・axios
